# 📚 BookLabs - Sistema de Biblioteca

## 📖 Sobre o Projeto

O **BookLabs** é um sistema simples de gerenciamento de biblioteca desenvolvido em **Python** para fins acadêmicos. O projeto tem como objetivo colocar em prática os principais conceitos da linguagem, especialmente operações de **CRUD (Create, Read, Update e Delete)**, manipulação de arquivos e organização de código em módulos.

Os dados da aplicação são armazenados localmente em um arquivo **JSON**, simulando um pequeno banco de dados.

---

## 🎯 Objetivos

* Praticar programação em Python;
* Aplicar o conceito de CRUD;
* Trabalhar com manipulação de arquivos JSON;
* Utilizar o módulo `os` para interação com o sistema operacional;
* Desenvolver um sistema organizado e modular;
* Atender aos requisitos da disciplina da faculdade.

---

## 🛠️ Tecnologias Utilizadas

* Python 3
* JSON
* Módulo `os`
* Terminal (CLI)

---

## ⚙️ Funcionalidades

* 📚 Cadastrar livros
* 📋 Listar todos os livros
* 🔍 Buscar livros por título ou ISBN
* ✏️ Editar informações de um livro
* 🗑️ Excluir livros
* 💾 Armazenamento dos dados em arquivo JSON

## ▶️ Como Executar

1. Clone este repositório:

```bash
git clone https://github.com/SEU-USUARIO/BookLabs.git
```

2. Entre na pasta do projeto:

```bash
cd BookLabs
```

3. Execute o programa:

```bash
python main.py
```
